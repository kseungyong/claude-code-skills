# Claude Code Custom Skills

Collection of custom skills for Claude Code, created by @kseungyong.

## Skills

- **explain-code** - Code explanation with visual diagrams and analogies
- **notebooklm-save** - Auto-save design documents to NotebookLM
- **plan-to-doc** - Auto-save implementation plans to docs/
- **ui-ux-pro-max** - UI/UX design intelligence with 67+ styles, color palettes, and component patterns

## Installation

Clone or download to your local Claude Code skills directory:

```bash
git clone https://github.com/kseungyong/claude-code-skills.git ~/.claude/skills
```

Or copy individual skill folders to `~/.claude/skills/<skill-name>/`.

## Usage

Skills are automatically available in Claude Code after installation. Invoke them using `/skill-name` command.
